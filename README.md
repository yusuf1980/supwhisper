# SupWhisper
A Windows app for offline transcription and subtitle generation.

This repository is used to distribute the Windows installer.

## Features:
- Transcribe audio or video files effortlessly right from your Windows PC.
- Supporting both CPU and NVIDIA GPU acceleration for optimal performance.
- Generate fast, high-quality transcriptions with GPU acceleration support.
- All transcribe processing happens locally on your device—no data is uploaded—making it perfect for sensitive content like interviews.
- Simply drag and drop files to start transcribing instantly.
- Supports 99+ languages with optional automatic language detection.
- Works with a wide variety of audio and video file formats.
- Edit, refine, or remove specific segments within your transcript.
- Save your transcription results for later editing.
- Subtitle generator.
- Export your work as .srt, .vtt, .txt, or .json files.
- Edit subtitle files (.srt and .vtt) with a built-in inline video player.
- Enjoy synchronized playback between audio, video, transcripts, and subtitles.
- Includes an inline player directly within the transcript editor for seamless editing.
- Monitor CPU and GPU usage with the built-in system monitor.
- Resume capability for longer media.
- Supports all Whisper models, with advanced models available for Pro users.
- Dark mode support for a more comfortable user experience.
